# Data-analytics  
  
## Stand-alone files:  
```
python graphCompare.py
python heatmap.py
python linear.py 
```
  
## How to run:  
```
perl run.pl <stockname>
```
  
"run.pl" will execute following files:  
 - predict-gaussian.py  
 - predict-linear.py  
 - predict-polynomial.py  
 - k-fold-gaussian.py  
 - k-fold-linear.py  
 - k-fold-polynomial.py
